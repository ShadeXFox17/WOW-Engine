# The WOW! Engine
## A 2.5 dimensional engine written in C#.

I love the LittleBigPlanet franchise. However, an issue with it is that it's hard to play with friends. So, why not make my own?
Using C# and Glfw, this engine is meant to be a (somewhat accurate) recreation of LittleBigPlanet's. I can't guarantee it'll be 1:1 but I sure as hell am gonna try.
